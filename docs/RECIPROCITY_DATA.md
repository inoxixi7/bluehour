# 胶片倒易率失效补偿参考表 (Reciprocity Failure Reference)

本文档列出了常见胶片的倒易率失效补偿参数。数据来源于官方技术数据表及摄影社区的长期测试经验。

> **注意**：倒易率失效受温度、批次和冲洗条件影响，以下数据仅供参考。在关键拍摄中，建议进行包围曝光。
