# 用户预设分步创建功能

## 概述
将创建新预设的流程改为分步向导模式，提供更清晰的用户体验。

## 功能特性

### 四步创建流程

#### 第一步：基本信息
- 预设名称（必填）
- 相机型号（可选）
- 镜头信息（可选）

#### 第二步：选择光圈
- Checkbox 多选框样式
- 显示所有可用光圈值（如 f/1.4, f/2, f/2.8 等）
- 显示已选择项数量

#### 第三步：选择快门
- Checkbox 多选框样式
- 显示所有可用快门速度
- 显示已选择项数量

#### 第四步：选择 ISO
- Checkbox 多选框样式
- 显示所有可用 ISO 值
- 显示已选择项数量

### 导航控制
- **上一步按钮**：在第 2-4 步显示，返回上一步
- **下一步按钮**：在第 1-3 步显示，进入下一步
- **保存按钮**：在第 4 步显示，完成创建

### 进度指示器
- 顶部显示 4 个步骤圆圈
- 当前步骤高亮显示
- 已完成步骤显示对号
- 步骤间用线条连接

## 技术实现

### 新增组件
- `CreatePresetStepper.tsx`：分步向导主组件

### 修改组件
- `UserPresetsManager.tsx`：集成新的分步组件

### 状态管理
- 使用 `useState` 管理当前步骤
- 表单数据在各步骤间共享
- 支持编辑已有预设（预填充数据）

## UI/UX 改进
1. **清晰的视觉层次**：每一步聚焦于一类信息
2. **进度可见性**：用户随时了解当前位置
3. **灵活导航**：可前后切换步骤
4. **即时反馈**：显示已选择项数量
5. **验证提示**：第一步验证必填项

## 使用方式
1. 在设置页面点击"创建新预设"
2. 按照分步向导填写信息
3. 可随时点击上一步修改
4. 最后一步点击保存完成创建
