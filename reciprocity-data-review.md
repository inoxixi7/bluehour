# 倒易律失效数据审查文档

## 概述
本文档包含应用中所有胶片的倒易律失效（reciprocity failure）补偿数据，用于长时间曝光计算。请审查数据的准确性。

## 数据来源与计算方法

### 胶片总数统计
- **总计**: 57种胶片配置
- **彩色负片 (C-41)**: 19种
- **黑白胶片**: 21种  
- **反转片 (E-6)**: 10种
- **电影胶片**: 3种
- **Foma系列**: 3种
- **其他**: 1种 (数码)

### 1. 曲线生成函数
```javascript
createPowerCurve(p) = t_corrected = Math.round(t_base^p)
```
其中 `t_base` 为基础曝光时间，`p` 为倒易律失效系数。

### 2. 基准时间点
```
BASE_SECONDS = [1, 2, 4, 8, 15, 30, 60, 120, 240, 480, 900, 1800, 3600]
```

## 需要验证的关键点

### 1. **超长曝光的合理性**
某些胶片在极长曝光时间（如1小时以上）的补偿值非常大，请确认：
- Kodak Tri-X 在3600s时补偿到7,566,158s (约2,102小时) 是否合理？
- Kodak Portra 在3600s时补偿到1,973,820s (约548小时) 是否合理？

### 2. **幂函数系数验证**
以下系数请确认是否符合厂商数据：
- Kodak Tri-X: p = 1.54 (倒易律失效严重)
- Kodak Portra: p = 1.35
- Kodak Gold: p = 1.4
- Ilford Delta 400: p = 1.41
- Ilford HP5 Plus: p = 1.31
- T-Max 100: p = 1.15 (倒易律失效轻微)

### 3. **特殊胶片数据**
以下胶片使用手动数据点而非公式：
- Fuji Acros II: 120秒内几乎无补偿
- Fuji Velvia 50: 倒易律失效剧烈
- Fuji Provia 100F: 128秒内无补偿
- Fomapan 系列: 中等倒易律失效

### 4. **数据一致性问题**
- Fomapan 100 和 Fomapan 400 使用相同曲线，这是否正确？
- Ilford Delta 100 和 FP4 Plus 使用相同曲线 (p=1.26)，是否合理？

### 5. **缺失的数据点**
手动数据胶片只提供了少量数据点，超出范围时使用线性外推。是否需要补充更多数据？

## 计算逻辑验证

### 插值算法
在两个数据点之间使用线性插值：
```
ratio = (t_input - t1) / (t2 - t1)
t_output = t_corrected1 + ratio * (t_corrected2 - t_corrected1)
```

### 外推算法
超出最后一个数据点时：
```
gradient = (last_corrected - prev_corrected) / (last_base - prev_base)
t_output = last_corrected + gradient * (t_input - last_base)
```

- 夜间长曝光摄影（星轨、星空）

- 建筑摄影长曝光
- 使用 ND 滤镜的日间长曝光

请根据实际使用场景验证数据的合理性。

## 已知问题与待改进

1. **超长曝光数据**: 3600秒以上的补偿值可能存在较大误差，建议实拍测试
2. **停产胶片**: 部分胶片已停产（如Fuji Pro 400H、多数Fujichrome系列），数据仅供参考
3. **批次差异**: 同型号胶片不同批次可能有轻微倒易律差异
4. **温度影响**: 当前数据基于常温(20°C)，极端温度下需额外补偿
