# 📱 Blue Hour - 風景・星空撮影プランニングアプリ

風景写真家やアウトドア写真家のために設計された包括的な撮影プランニングツールです。ブルーアワー、日の出・日没時刻の計算、露出計算、フィルム撮影の逆反則計算など、豊富な機能を備えたReact Nativeアプリです。

## ✨ 主な機能

### 📍 ブルーアワー・プランニング
- 任意の場所のブルーアワーの正確な時刻を検索・計算
- 日の出・日没時刻とブルーアワーの開始・終了時刻を表示
- お気に入りの撮影地を保存して素早くアクセス
- 現在地または手動での位置検索をサポート
- **ブルーアワーとは**：日の出前と日没後の短い時間帯で、空が深い青色を呈します。風景・建築写真に最適な「魔法の時間」です。

### 📸 露出計算機
- 露出三角形の計算：絞り・シャッター速度・ISO感度
- **EVロック機能**：基準EVを固定し、一つのパラメータを変更すると他が自動調整される
- **NDフィルター計算**：NDフィルターを使用した際のシャッター速度自動調整をサポート
- 計算結果から直接逆反則計算機へジャンプ

### 🎞️ 逆反則計算機（フィルム補正）
- **逆反則失効補正**：長時間露光でのフィルム感度低下を自動補正
- 11種類の主要フィルムタイプをサポート：Portra 400、Ektar 100、Provia 100F、Velvia 50など
- **内蔵タイマー機能**：補正後のシャッター速度でカウントダウン、バックグラウンド動作と通知をサポート
- 各フィルムのパラメータと特性の詳細な説明

### 🎨 ユーザープリセット管理
- カスタム撮影プリセット（絞り・ISO・ND組み合わせ）の保存
- プリセットの素早い適用と管理
- 撮影スタイルに合わせたパーソナライズ設定

### 🌍 多言語対応
- 中国語簡体字（デフォルト）
- 英語
- 日本語
- ドイツ語

### 🎭 ダーク/ライトテーマ
- システムテーマに従う
- ワンタップでライト/ダークモード切り替え

## 📖 撮影用語の説明

### ブルーアワーとは？

**定義**：ブルーアワーは日の出前と日没後の短い時間帯で、太陽が地平線下に沈むことで、散乱した光が空を深く鮮やかな青色にします。

**特徴**：
- 柔らかく均一な光線、コントラストが過度でない
- 空が深い青色を呈し、色温度が低い
- 環境光と人工光のバランスが良い
- 一般的に日の出前30分と日没後30分前後に発生

**撮影テーマに最適**：
- 都市の風景：夜景と空のバランスが良い
- 建築写真：柔らかな光が建物の輪郭を際立たせる
- 星空撮影：空がまだ完全に暗くなっていない時の前景との組み合わせ
- 海岸風景：静かで穏やかな光の雰囲気

---

### 逆反則律とは？

**定義**：逆反則律は、露光量（Exposure）= 照度（Illuminance）× 時間（Time）の関係を指します。理論上、照度を半分にし、時間を2倍にすれば、露光量は変わりません。

**公式**：
```
E = I × t
```
- E：露光量（Exposure）
- I：照度（Illuminance）
- t：露光時間（Time）

**例**：
- 絞りF/8、シャッター速度1/125秒、ISO 100
- 絞りをF/11に変更（光量1段分減少）
- シャッター速度を1/60秒に変更（時間を2倍に）
- 最終的な露光量は変わらない

これは**露出三角形**（絞り・シャッター速度・ISO）を調整する際の基本原理です。

---

### 逆反則失効とは？

**定義**：フィルム撮影において、露光時間が一定の閾値（一般的に1秒以上）を超えると、**逆反則律が成立しなくなり**、実際の露光量が理論値より少なくなる現象です。

**原因**：
- フィルム乳剤の化学的性質
- 長時間露光下で感光粒子の反応効率が低下
- 異なるフィルムタイプで失効の程度が異なる

**影響**：
- **露出不足**：通常のメーター通りに撮影すると、フィルムが暗くなりすぎる
- **色ずれ**：カラーフィルムでは色温度の変化や彩度の低下が発生する可能性
- **コントラスト変化**：シャドウ部のディテールが失われる

**解決方法**：
このアプリの**逆反則計算機**は、各フィルムタイプに基づいて自動的に補正係数を計算し、実際に必要なシャッター速度を提供します。

**例**：
- Kodak Portra 400フィルムを使用
- メーターで測定したシャッター速度：30秒
- 逆反則失効を考慮すると、実際には**52秒**露光する必要がある
- このアプリが自動計算し、タイマー機能も提供

## 📥 ダウンロードと使用

### インストール方法

1. **Android**：APKファイルをダウンロードしてインストール（近日公開予定）
2. **iOS**：App Storeからダウンロード（近日公開予定）
3. **開発者モード**：プロジェクトをクローンして`npm install && npm start`を実行

### 使用ヒント

**風景写真家向け**：
1. 事前に撮影地のブルーアワー時刻を検索し、到着時間を計画
2. 露出計算機を使用して、現場の光条件に基づいて絞りとシャッター速度を素早く計算
3. よく使う設定（例：風景撮影のF/11 + ISO 100）をプリセットとして保存

**フィルム写真家向け**：
1. 長時間露光撮影時（特に夜景・星空）には、必ず逆反則計算機を使用
2. 使用するフィルムタイプ（例：Portra 400やProvia 100F）を選択
3. メーターで測定したシャッター速度を入力し、補正後の時間を取得
4. 内蔵タイマーを使用して正確な露光時間を制御
