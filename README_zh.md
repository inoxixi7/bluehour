# 📱 Blue Hour - 风景与星空摄影规划应用

专为风光摄影师和户外摄影师设计的综合摄影规划工具。这款 React Native 应用提供了丰富的功能，包括蓝调时刻计算、日出日落时间、曝光计算以及胶片摄影的倒易律计算。

## ✨ 主要功能

### 📍 蓝调时刻规划
- 搜索并计算任意地点的精确蓝调时刻
- 显示日出日落时间以及蓝调时刻的起止时间
- 保存您喜爱的摄影地点以便快速访问
- 支持当前位置或手动位置搜索
- **什么是蓝调时刻？**日出前和日落后的短暂时段，此时天空呈现深邃的蓝色。这是风光和建筑摄影的"魔法时刻"。

### 📸 曝光计算器
- 曝光三角计算：光圈、快门速度、ISO
- **EV 锁定功能**：锁定基准 EV 值，改变一个参数其他自动调整
- **ND 滤镜计算**：支持使用 ND 滤镜时的快门速度自动调整
- 从计算结果直接跳转到倒易律计算器

### 🎞️ 倒易律计算器（胶片补偿）
- **倒易律失效补正**：自动补偿长时间曝光时的胶片感光度损失
- 支持 11 种主流胶片类型：Portra 400、Ektar 100、Provia 100F、Velvia 50 等
- **内置计时器功能**：使用补正后的快门速度倒计时，支持后台运行和通知
- 详细说明每款胶片的参数和特性

### 🎨 用户预设管理
- 保存自定义摄影预设（光圈、ISO、ND 组合）
- 快速应用和管理预设
- 为您的拍摄风格个性化设置

### 🌍 多语言支持
- 简体中文（默认）
- 英语
- 日语
- 德语

### 🎭 深色/浅色主题
- 跟随系统主题
- 一键切换浅色/深色模式

## 📖 摄影术语解释

### 什么是蓝调时刻？

**定义**：蓝调时刻是日出前和日落后的短暂时段，此时太阳位于地平线以下，散射光将天空染成深邃而鲜艳的蓝色。

**特点**：
- 柔和均匀的光线，对比度不会过强
- 天空呈现深蓝色调，色温较低
- 环境光与人工光线达到良好平衡
- 通常发生在日出前和日落后约 30 分钟

**最佳拍摄主题**：
- 城市风光：夜景与天空的良好平衡
- 建筑摄影：柔和的光线突出建筑轮廓
- 星空摄影：天空尚未完全变暗时结合前景
- 海岸风光：平静安详的光线氛围

---

### 什么是倒易律？

**定义**：倒易律指出曝光量 = 照度 × 时间。理论上，如果将照度减半并将时间加倍，曝光量保持不变。

**公式**：
```
E = I × t
```
- E：曝光量（Exposure）
- I：照度（Illuminance）
- t：时间（Time）

**示例**：
- 光圈 F/8，快门速度 1/125s，ISO 100
- 改为光圈 F/11（减少 1 档光线）
- 改为快门速度 1/60s（时间加倍）
- 最终曝光量保持不变

这是调整**曝光三角**（光圈、快门速度、ISO）时的基本原理。

---

### 什么是倒易律失效？

**定义**：在胶片摄影中，当曝光时间超过特定阈值（通常超过 1 秒）时，**倒易律不再成立**，实际曝光量会小于理论值。

**原因**：
- 胶片乳剂的化学性质
- 长时间曝光时感光粒子反应效率降低
- 不同胶片类型的失效程度不同

**影响**：
- **曝光不足**：按测光表正常拍摄会使胶片过暗
- **色彩偏移**：彩色胶片可能出现色温变化或饱和度降低
- **对比度变化**：阴影细节可能丢失

**解决方案**：
本应用的**倒易律计算器**会根据每种胶片类型自动计算补正系数，并提供实际需要的快门速度。

**示例**：
- 使用 Kodak Portra 400 胶片
- 测光快门速度：30 秒
- 考虑倒易律失效，实际需要曝光 **52 秒**
- 本应用自动计算并提供计时器功能

## 📥 下载与使用

### 安装方式

1. **Android**：下载并安装 APK 文件（即将推出）
2. **iOS**：从 App Store 下载（即将推出）
3. **开发者模式**：克隆项目并运行 `npm install && npm start`

### 使用技巧

**风光摄影师**：
1. 提前搜索拍摄地点的蓝调时刻并规划到达时间
2. 使用曝光计算器根据现场光线条件快速计算光圈和快门速度
3. 将常用设置（如风光摄影的 F/11 + ISO 100）保存为预设

**胶片摄影师**：
1. 长时间曝光（尤其是夜景和星空摄影）时务必使用倒易律计算器
2. 选择您的胶片类型（如 Portra 400 或 Provia 100F）
3. 输入测光表测得的快门速度并获取补正后的时间
4. 使用内置计时器精确控制曝光时间
