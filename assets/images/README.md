# 图片资源文件夹

存放应用所需的所有图片资源。

## 必需文件

请创建以下图片文件：

1. **icon.png** (1024x1024)
2. **splash.png** (1242x2436)
3. **adaptive-icon.png** (1024x1024)
4. **favicon.png** (48x48)

## 设计建议

### 主题色
- **蓝色**: #3498db (蓝调时刻)
- **金色**: #f39c12 (黄金时刻)
- **深色**: #1a1a2e (背景)

### 图标元素
- 相机符号
- 日出/日落
- 时钟/时间线
- 光圈符号

## 快速开始

在开发阶段，可以先使用纯色占位符：
- 使用在线工具生成简单图标
- 或从设计资源网站下载模板
